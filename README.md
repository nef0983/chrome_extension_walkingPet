# 散步寵物

一個有趣的 Chrome 擴充功能，在你瀏覽網頁時放置一個會自主移動的調皮寵物。當你移動滑鼠靠近它時，它會驚慌逃跑！

## 功能特點

- 🐱 **動畫寵物** - 在網頁上放置一個會自主漫步的寵物角色
- 🏃 **逃跑AI** - 當滑鼠接近時，寵物會快速溜走
- 🎯 **智能運動** - 寵物會隨機移動，碰到邊界時會反彈
- ⚡ **動態速度** - 逃跑時速度更快，正常狀態下緩慢漫步
- 🎨 **平滑動畫** - 流暢的移動過渡和旋轉效果

## 安裝方式

### 方式一：開發者模式安裝（推薦用於開發）

1. 打開 Chrome 瀏覽器，在位址列輸入 `chrome://extensions/`
2. 啟用右上角的 **開發人員模式**
3. 點擊 **載入未封裝的擴充功能**
4. 選擇此專案的資料夾
5. 擴充功能將出現在你的工具列上

### 方式二：直接安裝 .crx 檔案

如果你已經有打包好的 `.crx` 檔案：

1. 打開 Chrome 瀏覽器，在位址列輸入 `chrome://extensions/`
2. 將 `.crx` 檔案**拖拉**到 `chrome://extensions/` 頁面
3. 在彈出的確認對話框中點擊 **新增擴充功能**
4. 擴充功能將自動安裝並出現在你的工具列上

## 使用方式

1. 擴充功能啟用後，在任何網頁上都會看到一個可愛的寵物角色
2. 寵物會自動在頁面上漫步移動
3. 將滑鼠指向寵物，它會立刻驚慌逃跑！
4. 寵物會在邊界處反彈，避免離開螢幕

## 專案結構

```
chrome_extension/
├── manifest.json          # 擴充功能配置文件
├── index.html             # 彈出視窗的 HTML 檔案
├── popup/
│   └── popup.js           # 彈出視窗的 JavaScript 邏輯
├── scripts/
│   └── content.js         # 寵物運動邏輯與交互
├── images/
│   └── pet.gif            # 寵物角色 GIF 動畫（需自行添加）
└── README.md              # 本檔案
```

## 核心機制

### 寵物移動 (`movePet` 函式)

- 每 50ms 執行一次，確保流暢的動畫
- 隨機晃動使寵物看起來很自然
- 動態阻力系統：逃跑後速度會逐漸恢復正常

### 速度控制

- **正常速度** (`normalMaxSpeed = 3`) - 悠閒漫步
- **逃跑速度** (`escapeMaxSpeed = 25`) - 快速逃脫
- 速度衰減機制 (`currentMaxSpeed *= 0.95`) - 逐漸回到正常狀態

### 邊界檢查

- 寵物會在視窗邊界處反彈
- 不會超出螢幕範圍

### 逃跑邏輯

- 當滑鼠懸停時觸發 `mouseover` 事件
- 寵物朝著滑鼠相反方向快速逃跑
- 速度限制臨時提升至最大

## 自定義寵物角色

要使用你自己的寵物圖像：

1. 在專案中建立 `images` 資料夾
2. 將你的 GIF 或 PNG 圖片放入該資料夾，命名為 `pet.gif`
3. 在 `manifest.json` 中的 `web_accessible_resources` 中添加圖片資源
4. 重新載入擴充功能

### 推薦圖片規格

- **格式**: GIF 或 PNG
- **背景**: 透明背景
- **大小**: 80×80 像素（可在 content.js 中調整）
- **風格**: 可愛角色或寵物

## 權限說明

此擴充功能需要以下權限：

- **tabs** - 存取分頁資訊
- **activeTab** - 存取當前活躍分頁
- **host_permissions** - 在所有 HTTP/HTTPS 網站上執行

## 開發與修改

若要修改此擴充功能：

1. 編輯 `scripts/content.js` 中的常數來調整寵物行為
2. 修改 `pet.style.cssText` 来改變外观（大小、透明度等）
3. 在 `chrome://extensions/` 頁面上點擊 **重新載入**
4. 刷新目標網頁以看到最新更改

### 調整寵物大小

```javascript
pet.style.cssText = `
  position: fixed;
  width: 100px;  // 改變這個值
  z-index: 999999;
  ...
`;
```

### 調整速度

```javascript
const normalMaxSpeed = 5; // 提高正常速度
const escapeMaxSpeed = 30; // 提高逃跑速度
```
